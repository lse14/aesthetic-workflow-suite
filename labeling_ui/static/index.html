服务设置（server）

控制 WebUI 服务监听地址。修改后需重启服务生效。

抽样设置（sampling）

控制每次取图时的重试、超时和最小分辨率过滤。
比例用于“下一张”抽样；点击“保存设置到 YAML”后会持久化到配置文件。

Danbooru 图源

使用 Danbooru API 抽样。推荐配置账号/API Key 以提升稳定性。

e621 图源

e621 建议必须填写可识别 User-Agent，并控制请求速率。

本地图源（local）

从本地目录/文件随机抽样。路径可多行填写。
当你修改本地路径、扩展名或新增文件后，点击此按钮刷新可抽样列表。

存储设置（storage）

下载或读取后的图像统一转存为 WebP，并写入本地数据库。

评分标准（可自定义）

每个维度都可设置“描述 + 1~5 分示例”。示例格式必须为每行 `分值:说明`，例如 `4:构图成熟，视线引导自然`。

快捷键设置（仅当前浏览器）

可自定义“按钮操作”的快捷键。避免使用 1~5 与 Ctrl+1~5（它们用于多维切换与打分）。
图站 API 使用说明（重点）
1. Danbooru:
- 推荐设置 `DANBOORU_USERNAME`、`DANBOORU_API_KEY`。
- 建议填写 `user_agent`，避免 403。
- `tags` 可填如: `1girl solo rating:s score:>20`。
2. e621:
- 必须填写可识别 `user_agent`（建议含用途+联系方式）。
- 推荐设置 `E621_LOGIN`、`E621_API_KEY`。
- 建议降低请求频率，避免风控。
3. 本地源:
- `paths` 每行一个目录/文件，支持多行。
- 修改后请点“重建本地索引”。
4. 生效规则:
- `host/port` 修改后需重启服务。
- 其余设置保存后会立即生效。

cmd 示例:
set DANBOORU_USERNAME=your_name
set DANBOORU_API_KEY=your_api_key
set E621_LOGIN=your_name
set E621_API_KEY=your_api_key
提示: 配置保存失败时请检查端口范围、路径权限、以及 e621 的 user_agent 是否为空。
快捷键: 多维模式 Ctrl+1~5 切维度、1~5 打分；单维模式 1~5 打分。Shift+Enter 保存并下一张（预览顺序）, Ctrl+Enter 仅保存, Backspace 上一张, X 跳过, Space 下一张。
